# 30sec-hero

30秒で世界を救え。  
超高速バトルで敵を倒し、レベルを上げ、装備を整え、制限時間内に魔王を撃破するハイスピードRPGです。

## 概要

**30sec-hero** は、「30秒」という極端に短い制限時間の中で冒険・戦闘・成長・補給・ボス撃破を行う、超高速テンポのRPGです。

プレイヤーはフィールドを駆け回り、ザコ敵を倒して一気にレベルアップし、町で回復や装備購入を行いながら、マップのどこかにいる魔王を倒すことを目指します。

フィールドでは常に時間が減り続け、30秒が経過すると世界は滅びます。  
一方で、町や城の中では時間が止まるため、プレイヤーは限られた時間をどう使うかを考える必要があります。

## コンセプト

このゲームのテーマは、**30秒のタイムマネジメントRPG** です。

通常のRPGではじっくり探索し、少しずつレベルを上げていきます。  
しかし本作では、すべてが超高速で進みます。

- 30秒以内に魔王を倒す
- 数秒で戦闘が終わる
- 数回の戦闘で一気にレベルアップ
- 町では時間が止まる
- お金を払えば時間を巻き戻せる
- レベルはステージごとにリセット
- 装備や一部のお金は持ち越し可能

「急げ、でも焦るな」  
そんな緊張感と判断力が求められるゲームです。

## ゲームループ

基本的な流れは以下の通りです。

1. ステージ開始と同時に30秒のカウントダウンが開始
2. フィールドを探索し、ザコ敵と戦ってレベルアップ
3. 町や城に入り、時間を止めて回復・買い物・情報収集
4. 必要に応じてイベントやクエストを解決
5. 装備やレベルを整えて魔王に挑戦
6. 制限時間内に魔王を倒せばステージクリア
7. 30秒が経過するとゲームオーバー

## 主なシステム

### 30秒カウントダウン

ステージ開始時に30秒の制限時間が与えられます。

フィールド上や戦闘中はリアルタイムで時間が減少します。  
30秒が経過すると世界が滅び、ゲームオーバーになります。

ただし、町や城の中では時間が完全に停止します。  
プレイヤーは安全地帯で戦略を練ることができます。

### 女神像による時間巻き戻し

町にある女神像に祈ることで、制限時間を30秒まで巻き戻すことができます。

ただし、巻き戻しにはお金が必要です。  
さらに、同じステージ内で巻き戻しを行うたびに費用が増加します。

例：

- 1回目：10G
- 2回目：100G
- 3回目：300G
- 以降もさらに高額化

時間を買うか、装備を買うか。  
お金の使い道も重要な判断になります。

### 超高速バトル

フィールド移動中にランダムエンカウントが発生します。

戦闘は非常に短く、数秒で決着します。  
基本的には自動戦闘、またはシンプルな突撃バトル形式を想定しています。

ザコ敵を倒すことで経験値とゴールドを獲得できます。  
1ステージ内でLv1からLv10以上まで一気に成長するような、超インフレ気味のバランスです。

また、戦闘からはいつでも逃走可能です。  
時間を失うリスクと、経験値・ゴールドを得るリターンの判断が重要になります。

### 町・城システム

町や城は安全地帯です。  
中にいる間はカウントダウンが停止します。

町では以下のような行動ができます。

- 料理屋で食事をして体力を全回復
- ショップで武器・防具・アイテムを購入
- NPCから攻略のヒントを聞く
- 傭兵を雇う
- ギミック解決に必要なNPCと会話する
- 女神像で時間を巻き戻す

町でどれだけ準備するかが、魔王討伐の成功率を大きく左右します。

### 装備とアイテム

ショップでは以下のような装備やアイテムを購入できます。

- 武器
- 盾
- 兜
- 防具
- 薬草
- 自動回復アイテム
- 一時強化アイテム

レベルはステージごとにリセットされますが、装備は次のステージへ持ち越せます。  
そのため、長期的な成長は装備によって行います。

### イベント・クエスト

ステージによっては、魔王へたどり着くためにイベントの解決が必要になります。

例：

- 壊れた橋を木こりに直してもらう
- 道を塞ぐ岩をどかす
- NPCを救出して通行ルートを開放する
- 傭兵を仲間にして戦闘を有利にする
- 特定の敵を倒して鍵を入手する

すべてのイベントをこなす必要はありません。  
しかし、効率よく進めることで攻略が有利になります。

## ローグライト要素

ステージをクリアすると、そのステージ内で上げたレベルはリセットされます。

つまり、次のステージでは再び初期レベルからスタートします。

ただし、以下の要素は持ち越されます。

- 武器
- 防具
- 一部のお金
- 一部のアイテム

これにより、毎ステージごとに短期的なレベリングを行いながら、装備による長期的な強化も楽しめる設計になっています。

## ゲームの魅力

- 30秒という圧倒的なスピード感
- RPGの成長要素を短時間で味わえる
- 時間・お金・装備・ルート選択の戦略性
- 何度も挑戦したくなるステージ攻略
- 短時間で遊べるテンポの良さ
- 装備持ち越しによる継続的な成長
